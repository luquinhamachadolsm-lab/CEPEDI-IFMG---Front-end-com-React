# Gerenciador de tarefas - Tarefinhas ✔️
O projeto final do curso, um gerenciador capaz de salvar tarefas, editar, e as marcar como concluídas.
Possui as opções de tema claro e escuro, além de outras abas extras.
Possui a capacidade de salvar as tarefas mesmo depois de se fechar a aba do navegador,

Feito em React.
